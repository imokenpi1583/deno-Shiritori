# deno-Shiritori

## 概要

2人対戦のしりとりゲームです。3人目以降の接続は観戦が可能です。

## 実装した機能

### 1.対戦機能

- **2人対戦・観戦モード:**
  最初に接続した2名が自動的に「プレイヤー」としてマッチングされ、先攻・後攻がランダムに決定します。3人目以降の接続者は自動的に「観戦者モード」となり、リアルタイムで試合の進行を追うことができます。
  **手番管理（ターン制）:** データベース（Deno
  KV）と同期した手番判定を行い、自分のターン以外は入力フォームと送信ボタンが自動で無効化（disabled）されます。

### 2.正規化処理

- **連続伸ばし棒（長音記号）の圧縮:**
  「モンブランーーー」や「ラジーーオ」のように、伸ばし棒を連打して判定をすり抜けようとする入力を、正規表現
  `/ー+/g` によって1つの「ー」へ圧縮します。

* **末尾・接続文字抽出:**
  「ラジオー」のように末尾が長音記号で終わる単語の場合、自動的に1文字戻って「オ」を次の接続文字として認識します。
  小文字（ぁ、ゃ、ゅ、ょ、っ等）で終わる場合は、自動的に大文字（あ、や、ゆ、よ、つ等）に正規化して次のプレイヤーに引き継ぎます。

* **表記揺れを考慮した重複チェック:**
  過去に登場した単語の履歴をすべて「平仮名変換＋長音圧縮」した状態で保存。カタカナ・ひらがなの表記揺れがあっても、同一単語であれば重複として弾きます。

## 開発環境・技術スタック

- **ランタイム:** Deno
- **データベース:** Deno KV (状態管理・永続化)
- **通信プロトコル:** WebSocket
- **フロントエンド:** HTML5, CSS3, プレーン JavaScript (Vanilla JS)
